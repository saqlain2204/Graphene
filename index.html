// how it works

From folder to insight
in four steps

01 / drop
Drop a folder
Drag your project folder onto the window — or click to browse. Works with any size project.
02 / parse
Parse & resolve
The engine reads every file, extracts symbols, and resolves all cross-file dependencies.
03 / render
Graph appears
A live force-directed graph renders with nodes for files and typed edges for connections.
04 / explore
Explore freely
Click, zoom, filter, and search. Open any file inline. Understand the whole architecture.
